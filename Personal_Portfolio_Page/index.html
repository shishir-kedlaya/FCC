This pen was written for the article CSS-only dark mode.

Using the :checked pseudo-class selector, one can style descendants and subsequent siblings of a checkbox when it is in checked state.

Having both a checkbox for enabling dark mode and an element containing all other content on the page on the same level of the DOM tree allows us to offer a CSS-only dark mode of a web page.
